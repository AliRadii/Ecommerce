# E-commerce Platform Analysis

![Python](https://img.shields.io/badge/python-3.11-blue)
![Pandas](https://img.shields.io/badge/pandas-1.6-green)
![Scikit-learn](https://img.shields.io/badge/scikit--learn-1.2-orange)

This project analyzes an E-commerce dataset and builds **machine learning models** for tasks such as customer segmentation, sales prediction, and product recommendations. It demonstrates **data analysis, preprocessing, and predictive modeling** on real-world e-commerce data.

## Dataset
The dataset contains information on customer transactions, product categories, purchase history, and sales metrics.

## Features
- CustomerID, Gender, Age, Location, ProductID, ProductCategory, Quantity, Price, PurchaseDate, TotalAmount

## Usage
Open `Ecommerce.ipynb` in Jupyter Notebook to explore the analysis, build models, and visualize insights.
