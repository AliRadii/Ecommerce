This project analyzes an **E-commerce customer dataset** and builds predictive models to understand customer behavior and spending patterns. It demonstrates **data analysis, preprocessing, and machine learning** techniques on real-world data.

## Dataset
The dataset contains information about customers, their interactions with the platform, and their spending habits.

## Features
- **Email** — Customer email ID (identifier)  
- **Address** — Customer address  
- **Avatar** — Profile image or avatar info  
- **Avg. Session Length** — Average time spent per session on the platform  
- **Time on App** — Time spent using the mobile app  
- **Time on Website** — Time spent on the website  
- **Length of Membership** — Duration of membership (in years)  
- **Yearly Amount Spent** — Total yearly spending by the customer (target for prediction)

## Usage
Open `Ecommerce.ipynb` in Jupyter Notebook to explore the data, perform analysis, and build predictive models.
